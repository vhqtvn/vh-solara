## Scope

Two themes.

### 1. dist-build hybrid embed model (P1-WEB-025 / 027 / 028)

- SPA builds to the **gitignored** staging dir `web/dist-build/` (not `pkg/web/dist/`).
- A self-contained fallback `pkg/web/dist/index.html` (no `/assets` or `/sw.js` references) stays **tracked** so a cold-clone `go build` / `go test` compiles with no frontend build.
- Embed-producing targets — Makefile, `release.yml`, `scripts/fixture-web.sh`, `Dockerfile`, `Dockerfile.e2e` — **materialize** staged assets (`web/dist-build/*` → `pkg/web/dist/`) immediately before `go build`, via the shared `web/scripts/materialize.sh`.
- Net effect: `make web` leaves `git status` clean; the binary embeds the real SPA only when a materializing target runs.

### 2. Chat reveal-gate + scroll/UX polish (P1-WEB-020 / 021 / 022 / 023)

- Transcript reveal gate: messages render only after full delivery (no partial-stream flicker).
- `messagesError` lifecycle handling.
- Mid-history scroll-anchor restore.
- e2e coverage for the reveal gate.
